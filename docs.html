Utah Democratic Party – Precinct Lookup

API Reference

← Map API Docs

Overview

The Utah Democratic Precinct Lookup API resolves a Utah mailing address into its corresponding Democratic voting precinct. It returns the precinct name, identifier, county, geocoded coordinates, GeoJSON boundary geometry, and a self-contained renderable HTML map — all in a single request.

The API is open, requires no authentication from callers, and supports CORS so it can be called from any browser, server, or automated agent.

Endpoint

GET /api/precinct

Base URL

https://utahdemprecinctmap.vercel.app/api/precinct

Parameters

Parameter Type Required Description Example
address string Required Utah street address. Accepts city name or ZIP code as zone. Abbreviations (St, Ave, N, S, SLC, etc.) are handled automatically. 350 N State St, Salt Lake City
geometry boolean Optional Include GeoJSON polygon geometry in response. Default: true geometry=false
html boolean Optional Include a self-contained renderable HTML map in the response. Can be embedded directly in an <iframe> or served as a standalone page. Default: true html=false

Example Request

GET /api/precinct?address=350+N+State+St,+Salt+Lake+City

Example Response

{
  "success": true,
  "data": {
    "precinct": {
      "name":       "Salt Lake City 11",
      "precinctID": "SLC011",
      "vistaID":    "SLC011",
      "county":     "Salt Lake"
    },
    "address": {
      "input":    "350 N State St, Salt Lake City",
      "matched":  "350 N STATE ST, SALT LAKE CITY",
      "location": {
        "lat": 40.77657,
        "lng": -111.88836
      }
    },
    "geometry": {
      "type": "Polygon",
      "coordinates": [ [ [...], [...], ... ] ]
    },
    "html": "<!DOCTYPE html>...</html>"
    // Self-contained HTML page with Leaflet map, precinct boundary drawn,
    // and address pin. Renderable directly in an iframe or browser tab.
  }
}

Error Responses

HTTP StatusCauseResolution
400 Missing address parameter Include ?address=... in the request
404 Address not geocoded, or no precinct found at that location Try adding a ZIP code or full city name; confirm address is in Utah
500 Upstream service error (UGRC or ArcGIS) Retry the request

Rendering the HTML Map

The html field in the response is a complete, self-contained HTML document. You can render it in two ways:

As an iframe (srcdoc)

<iframe srcdoc="..." width="100%" height="500" frameborder="0"></iframe>
When injecting into srcdoc, HTML-encode the string first (replace & → &amp;, " → &quot;, etc.).

As a Blob URL (JavaScript)

const blob = new Blob([data.html], { type: 'text/html' });
const url  = URL.createObjectURL(blob);
iframe.src = url;

Tips for Accurate Results

  • Include a city name or ZIP code — the geocoder needs a zone to narrow the result
  • Street abbreviations (N, S, E, W, St, Ave, Blvd, Dr) are all accepted
  • City abbreviations like SLC are not resolved — use Salt Lake City or the ZIP
  • Apartment/unit numbers can be omitted
  • PO Boxes are not supported — use a physical street address

For Automated Agents & LLMs

🤖 Machine-Readable API Summary

This API takes a Utah mailing address as plain text and returns structured precinct data. No API key or authentication header is required. All requests are GET. The response is always application/json.

Canonical Endpoint

https://utahdemprecinctmap.vercel.app/api/precinct?address={address}

Minimal Request (metadata only, no geometry/html)

GET /api/precinct?address=350+N+State+St+84114&geometry=false&html=false

Full Request (all data including renderable map)

GET /api/precinct?address=350+N+State+St,+Salt+Lake+City

Key fields to extract from response

Field pathDescription
data.precinct.nameHuman-readable precinct name
data.precinct.precinctIDShort precinct code (e.g. SLC011)
data.precinct.countyCounty name
data.address.locationGeocoded { lat, lng }
data.geometryGeoJSON Polygon of precinct boundary
data.htmlSelf-contained HTML map page (Leaflet)

Failure detection

Check response.success === false and read response.error for a plain-English message. HTTP status will be 400, 404, or 500 on failure.

Address formatting guidance for LLMs

When constructing the address parameter from user-provided data, prefer the format:
{house number} {street}, {city} — e.g. 123 Main St, Provo
or {house number} {street} {zip} — e.g. 123 Main St 84601.
Always URL-encode the value. Spaces may be encoded as + or %20.

Data Sources

Precinct boundaries are sourced from the Utah AGRC VistaBallotAreas layer maintained by UGRC and the Utah Lieutenant Governor's Office. Address geocoding is powered by the UGRC Web API. Boundaries are updated periodically following annexations and redistricting.
